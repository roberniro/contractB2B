# 2023년 1학기 건설환경캡스톤디자인 1조 IT프로젝트
<br/>

## 프로젝트 소개
### 프로젝트명
- 건설 INFO

### 프로젝트목적
- 건설 공사 및 용역업체 정보 공개, 원청 업체와 민원 상담 통한 건설업 투명성 및 시민 편의 증진

### 주요 기능
- 원청, 용역업체간 견적 발송
- 원청의 공사 상태 관리, 용역업체 평점 등록
- 원청, 시민간 실시간 채팅 상담

### 개발 기간
- 2023.05.04 ~ 2023.06.08
